leetcode33-搜索旋转排序数组

Posted on 2020-04-05 In Leetcode题解

搜索旋转排序数组

  • 假设按照升序排序的数组在预先未知的某个点上进行了旋转。 例如，数组 [0,1,2,4,5,6,7] 可能变为 [4,5,6,7,0,1,2]。
  • 搜索一个给定的目标值，如果数组中存在这个目标值，则返回它的索引，否则返回 -1 。
  • 你可以假设数组中不存在重复的元素。你的算法时间复杂度必须是O(log n) 级别。

示例

    • 输入: nums = [4,5,6,7,0,1,2], target = 0
    • 输出: 4
    • 输入: nums = [4,5,6,7,0,1,2], target = 3
    • 输出: -1

O(log n)自然想到的是二分查找,怎么分就很关键了。像[1, 2, 3, 4, 5, 6, 7]大概可以分成两类：

  • [2, 3, 4, 5, 6, 7, 1]这种，nums[start] ≤ nums[mid],此时左边是有序的，如果nums[start] ≤ target < nums[mid],则在前面查找，否则去后面。
  • [6, 7, 1, 2, 3, 4, 5]这种，nums[start] > nums[mid],此时右边是有序的，如果nums[mid] < target ≤ nums[end],则在后面查找，否则去前面。
